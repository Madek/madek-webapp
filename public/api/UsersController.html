Users

Get a collection of Users

/users [GET]

Optional Parameters

  • query String The search query to find matching users
  • exclude_group_id Integer The id of the group to exclude the members from the result

Examples

Assuming that Franco is member of group_id 1 Request {} Response [{"id":1,"name":"Sellitto, Franco"},{"id":2,"name":"Pape, Sebastian"}]
Request {"query": "franco"} Response [{"id":1,"name":"Sellitto, Franco"}]
Request {"exclude_group_id": 1} Response [{"id":2,"name":"Pape, Sebastian"}]
